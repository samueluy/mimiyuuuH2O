# mimiyuuuH2O
A Discord bot where in mimiyuuuh reminds you to drink your water, bhie!

Quick, one sitting project left with some bugs

## Requirements
* Discord Py
* FFmpeg
```sh
pip install discord.py; sudo apt update && sudo apt install ffmpeg
```

## Installation
```sh
pip install discord.py; sudo apt update && sudo apt install ffmpeg
```

## Usage
```sh
python mimiyuuuH2O.py
```
